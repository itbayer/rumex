Rumex bekommt eine (g)VIM Unterstützung

Da ich hauptsächlich mit (G)VIM an Rumex arbeite und hier schon ein paar .vim Schnippsel entstanden sind, wurden diese jetzt auch eingebaut. Die Erweiterung besteht zur Zeit nur aus ein paar Kurztasten und einem GVIM Menü.

Rumex’s RSS Feed hat nun auch ein Datum bekommen

Der Newsreader Lifera hatte, durch das fehlende XML TAG <pubDate></pubDate>, immer wieder Probleme mit der Anzeige der Rumex News. Rumex rss.rx0x wurde darum um das Feld | Datum: erweitert. Das passende Datum bekommt man über die Shell mit dem Befehl date -R, bzw. unter vim mit :r!date -R<CR>kJ.

rumex bekommt eine Bildbearbeitung

rumex bekommt eine sitemap.xml

Mit einer sitemap.xml erleichtert man den Suchmaschinen das finden einzelner Seite. Man kann auch sagen sie sitemap.xml ist eine Liste der Seiten die sich auf der Webseite befinden.

Natürlich braucht rumex auch so einen Datei. Erstellt wird die sitemap mit make sitemap oder bei make all bzw. make online. .ersteckt markierte Dateien *.md?v werden natürlich nicht aufgenommen.

rumex wird auf einen nicht github.com Server installiert

rumex funktioniert auch auf einen nicht github.com Server. Eine Beschreibung wie man den Baukasten dort installiert findet man auf der Seite http://rumex.it-bayer.de.

rumex bekommt die RSS Feed Funktion

In den rumex Baukasten wurde eine RSS Feed Funktion eingebaut. Die Einträge werden dabei auch aus einem pandoc Dokument gelesen und nach rss.xml geschrieben.

Zusätzlich ist diese Datei auch als rss.html unter http://www.it-bayer.de/rumex/rss.html verfügbar.
