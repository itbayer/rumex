Start

Dies ist ein Versuch mit ein paar Skripten eine Internetseite zu erstellen und zu verwalten. Eingesetzt wird dabei die Sprache markdown die dann von pandoc in das HTML Format gebracht wird.

Die Seite bzw. die Entwicklung liegt auf github.com, der aktuelle Stand kann als ZIP Datei herunter geladen werden.

Aktuell wird dieser Baukasten1 auch auf der Seite http://www.it-bayer.de verwendet.
