Start

Dies ist ein Projekt das mit pandoc's markdown und ein paar Skripten eine Internetseite erstellt und verwalten.

Gehostet werden kann das System auf github.com wo auch diese Seite zu finden ist aber auch auf Servern mit git Unterstützung oder mit einem einfachen FTP Zugang kann dieses System verwendet werden.

  • Beispiel der Seite auf einem Server mit git Unterstützung
  • Beispiel der Seite auf einem Server mit FTP Zugang

Der aktuelle Stand dieses Baukastens kann als ZIP Datei herunter geladen werden.

Aktuell wird dieser Baukasten1 auch auf der Seite http://www.it-bayer.de verwendet.

  1. Es kann aber sein das hier nicht die aktuelle rumex Version verwendet wird.↩
