Update

Notiz Zettel

Notizen über die Schritte die bei einem Update gemacht werden sollten.

Befehl

make update VERSION

Schritte

  • Version runter laden und nach /tmp speichern.
  • Datei im TMP entpacken
  • Dateien kopieren
    • .bin
      • alle Dateien
    • css
      • alle Dateien außer rumex.png Logo
    • .inc/default
      • allen neuen Vorgaben
