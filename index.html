IT Bayer’s rumex (github.com Version)

Stefan Blechschmidt

2015-06-20

Rumex ist ein Projekt das mit pandoc's markdown und ein paar Skripten eine einfache Internetseite erstellt und verwalten. Diese Seiten können dann auf github.com, wo auch diese Seite zu finden ist, gehostet werden. Aber auch auf Servern mit git Unterstützung oder mit einem einfachen FTP Zugang können die Seite veröffentlicht werden.

Beispiele

  • Rumex auf github (diese Seite)
  • Rumex auf einem Server mit git Unterstützung
  • Rumex auf einem Server mit FTP Zugang
  • Rumex auf einem host1free.com Server mit git Unterstützung

Der aktuelle Stand dieses Baukastens kann als ZIP Datei holen.

Aktuell wird dieser Baukasten auch folgenden Seiten eingesetzt bzw. getestet.

  • http://www.it-bayer.de
  • http://www.byvs.de
  • http://www.hochfelder.de
  • http://www.auggenbach.de

Weblog Beschreibung

Letzte Bearbeitung: 2015-02-03

Diese kleine Erweiterung soll eine Art Blog innerhalb Rumex ermöglichen. Besser gesagt ist diese Erweiterung eher ein Weblog Funktion.

Der Unterschied zu den eigentlichen Rumex Seiten ist nicht sehr groß. Folgt nach einer h1 Überschrift ein div Tag mit der weblog Klasse wird der Text innerhalb dieses div Tags ausgeblendet und mit dem Schriftzug ... weiter lesen ersetzt. Die Überschrift und der Text ... weiter lesen bekommen eine Link Eigenschaft die den ausgeblendeten Text bei Anforderung einblendet.

Diese Funktion kann auch in den normalen Seiten verwendet werden.
