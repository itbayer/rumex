IT Bayer’s rumex (github.com Version)

Stefan Blechschmidt

2014-01-27

Dies ist ein Projekt das mit pandoc's markdown und ein paar Skripten eine Internetseite erstellt und verwalten.

Gehostet werden kann diese auf github.com wo auch diese Seite zu finden ist aber auch auf Servern mit git Unterstützung oder mit einem einfachen FTP Zugang kann dieses System verwendet werden.

  • Rumex auf github (diese Seite)
  • Rumex auf einem Server mit git Unterstützung
  • Rumex auf einem Server mit FTP Zugang
  • Rumex auf einem host1free.com Server mit git Unterstützung

Der aktuelle Stand dieses Baukastens kann als ZIP Datei herunter geladen werden.

Aktuell wird dieser Baukasten auch auf der Seite http://www.it-bayer.de und auf der Seite http://www.hochfelder.de verwendet.

Titelzeilen umgebaut

Am 27.01.2014 um 14:27 schrieb IT-Bayer

Der Umbau der Titelzeile wurde gemacht um die Verwendung durch pandoc, Autor und Datum, besser umzusetzen.

Es wird zwar nur mehr eine Überschrifszeile angezeigt. Dafür können die andere beider Zeilen für die Angaben des Autors bzw. des Datums verwendet werden. Diese beiden Angaben werden in der Online Version nicht angezeigt. Sie dienen lediglich für die META Angabe im HEAD Bereich der HTML Seite. Wir die Seite ausgedruckt werden diese Beiden Angabe jedoch wieder eingeblendet.

Rumex WebLog

Am 03.12.2013 um 12:25 schrieb IT-Bayer

Rumex hat eine neue Seite bekommen weblog.rx0s. Diese Seite dient nun als WebLog auf der Index Seite und ersetzt die alte WebLog Funktion auf der start.rx0s.

Ich verspreche mir dadurch ein wenig mehr Übersicht auf der index.html.

Wie auch die start.rx0s wird die weblog.rx0s in der index.html immer ganz oben gehalten1. Egal ob sich der Inhalt dieser Seite geändert hat.

Als Seitengerüst für die WebLog Seite kann das Standard Gerüst einer .rx0s Seite verwendet werden. Es muss jedoch die Überschrift2 mit dem Link aus der Seite genommen werden.

Das Schlüsselwort schnipp bestimmt ob die WebLog Einträge auch auf der index.html oder nur im Archiv, die eigentliche Seite weblog.html, angezeigt werden.

… ältere Weblog Einträge anzeigen

  1. Genauer gesagt wird die weblog.rx0s immer nach der start.rx0s eingebaut.↩

  2. Gemeint ist hier die Überschrift mit dem Link zur Seite die am Anfang der Seite in den Kommentar Klammern steht. Diese können übrigens drin bleiben.↩
