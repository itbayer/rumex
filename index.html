IT Bayer’s rumex

Der(Ein) HomePage Sand(Bau)kasten

github.com Version

Dies ist ein Projekt das mit pandoc's markdown und ein paar Skripten eine Internetseite erstellt und verwalten.

Gehostet werden kann diese auf github.com wo auch diese Seite zu finden ist aber auch auf Servern mit git Unterstützung oder mit einem einfachen FTP Zugang kann dieses System verwendet werden.

  • Beispiel der Seite auf einem Server mit git Unterstützung
  • Beispiel der Seite auf einem Server mit FTP Zugang

Der aktuelle Stand dieses Baukastens kann als ZIP Datei herunter geladen werden.

Aktuell wird dieser Baukasten auch auf der Seite http://www.it-bayer.de verwendet.

WebLog Layout auf der Startseite

Das Layout der Weblog Einträge auf der Startseite schaut nicht so schön aus. Hier muss noch nachgebessert werden.

Der Eintrag Datum Autor braucht noch einen grau Ton.

Bei der Übersicht der Einzelseiten gefällt mir auf einmal die Datumsangabe nicht mehr. Die sollte jetzt unter der Überschrift stehen und evtl. mit dem Schlüsselwort Stand: gekennzeichnet werden.
