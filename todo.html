tilda

Durch die tilda Erweiterung müssen zwei unterschiedliche vim Konfigurationen erstellt werden.

  1. Standard vim Konfiguration die für vim und gvim verwendet wird.
  2. gvim Konfiguration die für die Menü Darstellung zuständig ist.
    die Standard Datei wird automatisch nachgeladen.

tilda Beschreibung überarbeiten, Startdateien anpassen und evtl. als Kopiervorlage in .rumex/bin/ ablegen.

Kurztasten / Menü

In der Menü Erweiterung sollte eigentlich nur die Kurztasten abgebildet werden. Leider sind nicht alle Menübefehle als Kurztaste umgesetztm

curl

evtl. Umstellung von wget / wput auf curl

Download

curl -o datei -# http://link.de/datei

Durch den # Parameter wird ein einfacher Download Balken angezeigt.

Upload

curl -T "{datei1,date2}" -u user:password ftp://link.de
